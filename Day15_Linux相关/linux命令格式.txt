cd Python-3.7.3- 5.指定安装路径    - ./configure --prefix==/usr/local/python3- 6.编译    -  make && make install- 7.清除临时文件    - make clean- 8.建立软连接    - ​ln -s /usr/local/python3/bin/python3 /usr/bin/python3    - ln -s /usr/local/python3/bin/pip3 /usr/bin/pip3vim命令：1.启动vim    - vim 文件名2.vim的三种模式：    - 命令模式 - 按键代表命令 ---i或者a---> 编辑模式                          ---：或者/ ---> 末行模式        ~ G - 光标移到末行 g - 光标移到行首 500G - 移到500行        ~ hjkl -向左向上向下向右移动 - 10l 光标向右移到10下        ~ contrl + y / contrl + e - 移动一行        ~ contrl + f / contrl + b - 翻一页        ~ 0 - 光标到当前行首 / $ - 光标到当前行尾        ~ dd 删除一行 - 100dd - 删除100行 （都是从光标位置开始向下删除）        ~ yy 复制光标所在行/ 100yy - 从光标所在位置复制100行        ~ p 粘贴 / 5p - 复制5遍        ~ u - 撤销 / ctrl + r - 恢复    - 编辑模式 - 键入内容 ---Esc---> 命令模式    - 末行模式 - 搜索/配置/执行系统命令 ---Enter---> 命令模式        ~ set nu 显示行号        ~ set nonu 不显示行号        ~ set ts=4 设置tab的空格数为4        ~ syntax on - 显示高亮语法 - syntax off        ~ set ruler - 显示光标所在位置 - set norulerredis数据库- redis-server 启动服务- redis-server & 在后台启动服务进程操作相关命令：~ ps - process 查看进程~ Ctrl + z 把前台命令停下来放在后台运行~ kill - 结束进程~ jobs -
